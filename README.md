# Snippet-Hotkeys
Lightweight global hotkey snippet tool for Windows.

## Features

- Global hotkeys
- Token expansion
- Works in browsers (Gmail)
- Simple UI
- No installer required

## Example Tokens

{TODAY}
{NOW}
{NEXT_BUSINESS_DAY}
{NEXT_BUSINESS_DAY_DATE}

## Usage

Download the latest release from:

Releases → SnippetHotkeys-v1.zip

Run:

SnippetHotkeys.exe
